# 쇼핑 리스트 앱

한국어 쇼핑 리스트 웹 애플리케이션입니다. 항목 추가/삭제, 완료 체크, 필터링 기능을 제공합니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 아이템 완료 체크 / 체크 해제
- 완료된 항목 일괄 삭제
- 전체 / 미완료 / 완료 필터링
- localStorage를 이용한 데이터 영속성 (새로고침 후에도 유지)

## 실행 방법

```bash
npm start
```

브라우저에서 http://localhost:8765 로 접속합니다.

## 기술 스택

- HTML / CSS / Vanilla JavaScript
- Node.js (HTTP 서버)
